## Install
### Backend
```bash
uv venv
uv sync
```
### Frontend
```bash
npm install
```
## Run
### Backend
```bash
cd api/
```
```bash
python3 main.py
```
### Frontend
```bash
cd frontend/
```
```bash
npm run dev
```
